# ProxyHub Frontend Project

## Description
This is a professional, frontend-only React web application serving as a comprehensive landing page, dashboard, and API documentation portal. It perfectly mirrors a premium SaaS/proxy service visual style.

The project features a highly organized directory structure built for scalability, utilizing React, Vite, React Router, and Tailwind CSS. All components are reusable, and the application is fully responsive.

**Key Features:**
- Frontend-only CatProxies-style landing page UI
- Frontend-only CatProxies-style documentation and dashboard UI
- Static mock dashboard and pricing data
- Clean directory structure (`src/components/home`, `src/components/layout`, `src/data`)
- Backend integration pending (comments added where API calls should go)

## Setup & Run

To install dependencies:
```bash
npm install
```

To start the development server:
```bash
npm run dev
```

To build for production:
```bash
npm run build
```
