# ProxyHub API Documentation Frontend

## Description
This is a professional, frontend-only React web application serving as the API documentation and management interface for ProxyHub. It features a modern design built with Vite, React, React Router, and Tailwind CSS. The interface is fully responsive and production-ready.

**Note:** Frontend-only project, backend integration pending. This application currently uses mock/static content. Backend developers will need to integrate the real API endpoints later.

## How to run
To start the development server:
```bash
npm install
npm run dev
```

## How to build
To build the application for production:
```bash
npm run build
```
The output will be placed in the `dist` directory.
